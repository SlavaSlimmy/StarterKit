# StarterKit
### Installation:
```sh
$ cd app
$ npm install
$ bower install
```
### Development:

Build 'dev'
```sh
$ gulp dev
```

Build 'dev' and start "watch"
```sh
$ gulp
```

Build 'dev', start "watch" and open in browser (http://localhost:9000)
```sh
$ gulp open
```

### Production:
Build 'prod' in folder 'production'
```sh
$ gulp prod
```
